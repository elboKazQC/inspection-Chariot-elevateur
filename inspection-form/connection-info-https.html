🔒 Connexion HTTPS pour iOS📱 iOS Optimisé

🚨 Important pour iOS/iPhone

Cette version utilise HTTPS pour une meilleure compatibilité avec iOS. Vous devrez accepter le certificat lors de la première connexion.

Pour vous connecter à l'application depuis votre iPhone/iPad, utilisez ces adresses HTTPS dans Safari :

🔗 https://10.136.136.143:8443
🔗 https://172.21.160.1:8443

📋 Étapes pour iOS :

📶1. Même réseau WiFi

Assurez-vous que votre iPhone/iPad et cet ordinateur sont connectés au même réseau WiFi d'entreprise.

🔗2. Ouvrir le lien HTTPS

Dans Safari sur votre iPhone, cliquez sur l'un des liens HTTPS ci-dessus.

🛡️3. Accepter le certificat

iOS affichera un avertissement de sécurité. Cliquez sur "Avancé" puis "Continuer vers le site".

✅4. Tester la connexion

L'application d'inspection devrait maintenant se charger correctement avec toutes les fonctionnalités iOS.

🔧 Dépannage spécifique iOS :

  • 🚫 "Connexion non sécurisée" : Normal avec HTTPS local. Cliquez "Avancé" → "Continuer".
  • 📱 Page blanche sur iPhone : Désactivez le "Mode Privé" dans Safari et rafraîchissez.
  • 🔄 Application lente : Fermez les autres onglets Safari et redémarrez Safari si nécessaire.
  • ✍️ Signatures ne fonctionnent pas : Utilisez Safari (pas Chrome) pour une meilleure compatibilité tactile.
  • 📷 Problème avec l'appareil photo : HTTPS résout les problèmes d'accès à la caméra sur iOS.
  • 🔐 Certificat refusé : Allez dans Réglages > Général > À propos > Réglages de certificat et faites confiance au certificat.
